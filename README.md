<h1 align="center">🤝🏻 Portfolio 📖</h1>

<h3>Main technologies used 👷🏻‍♂️</h3>
<ul>
  <li>🔰 Vue.JS</li>
  <li>🧱 HTML</li>
  <li>🎨 CSS</li>
</ul>
